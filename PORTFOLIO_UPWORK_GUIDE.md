# 💼 PARTE 2: Copia Técnica para el Caso de Estudio de Upwork

Cuando subas este proyecto a tu **Portafolio de Upwork**, completá los campos del formulario con el siguiente texto diseñado para captar clientes de alto presupuesto:

### 🏷️ Título del Proyecto (Project Title)
*   **Opción A (Recomendada):** `Full-Stack Team Analytics & Offline-First Platform with Local AI`
*   **Opción B (Español):** `Plataforma Full-Stack de Análisis Táctico con IA Local y Sincronización Offline`

### 📝 Descripción del Proyecto (Project Description)

#### 🇺🇸 English (Copy & Paste)
```markdown
### 🥊 The Challenge
A professional boxing team needed a way to digitize tactical fight analysis. Existing software was either too expensive or required constant high-speed internet, which is unavailable in training gyms and venues. 

### 💡 The Solution
I designed and developed a robust, offline-first hybrid desktop application in just 1.5 weeks. The app enables coaches to paint tactical vector overlays on top of high-definition video, sync observations millisecond-by-millisecond, leverage local AI vision models, and generate high-fidelity PDF reports for elite managers.

### 🛠️ Tech Stack & Architecture
* **Frontend:** React, TailwindCSS, Fabric.js (HTML5 Vector Canvas), Lucide Icons.
* **Backend:** Node.js, Firebase Auth & Firestore.
* **Database & Sync:** Dexie.js (IndexedDB local transaction engine) with background sync.
* **AI Orchestration:** Local Ollama integration running LLaVA for offline computer vision frame analysis.
* **Reports:** jsPDF + html2canvas print-engine.

### 🏆 Key Technical Achievements
1. **Zero-Latency Canvas:** Seamless drawing of punch trajectories frame-by-frame synced with the video timeline.
2. **Offline-First Synchronization:** The application continues to read, write, and authorize sessions offline, syncing seamlessly in the background as soon as connectivity is detected.
3. **100% Local AI:** Structured JSON output from image classification models running 100% offline, keeping running costs at zero.
```

### 💡 La Solución

La aplicación permite a los entrenadores dibujar vectores tácticos directamente sobre video de alta definición, sincronizar notas milisegundo a milisegundo, automatizar métricas mediante modelos locales de IA y generar dossiers impresos profesionales en PDF.

--------------------------------------------------------------------------------
The Daneri Boxing Team required a professional tool to replace manual tactical coaching

--------------------------------------------------------------------------------

   Coaches needed to trace visual paths on top of HD video streams. 
   * Solution: Superimposed a transparent vector canvas using Fabric.js.

   * Solution: Implemented IndexedDB via Dexie.js for lightning-fast, offline 
     writes. Developed a transactional queue that detects connection recovery 
     and syncs data smoothly to Firebase Firestore.

   To prevent high cloud costs and API latency during matches.
   * Solution: Orchestrated local LLM vision models (LLaVA/Llama 3.2 Vision) inside 
     the application. 
   * Result: The offline coach "Einstein" automatically classifies actions from 
     video frames, outputting structured JSON metrics for analysis.

D. EXECUTIVE PDF GENERATION (jsPDF & html2canvas)
   Coaches must deliver professional printable reports to team managers instantly.
   * Solution: Programmed an export engine using html2canvas and jsPDF. 
   * Result: Compiles all visual charts, stats, and tactical frames into a beautiful, 
     custom-styled PDF dossier within 3 seconds.
